# Shoe Module Engineering Specification v1

## Статус

`research` — производственная генерация заблокирована.

## Назначение первой версии

Проверить полный процесс на одной правой колодке, одной модели Derby, одном размере и клеевой конструкции. Цель — получить 2D-детали верха, собрать пробный верх и выполнить затяжку на исходной колодке.

## Входной контракт

### Колодка

- `STL`, `OBJ`, `PLY` или `GLB`;
- единицы — миллиметры;
- поверхность без неконтролируемых разрывов;
- подтверждённая продольная ось;
- подтверждённая плоскость следа;
- контрольные ориентиры;
- измерения: длина, ширина пучков, обхват пучков, подъём, пяточный обхват, высота каблука.

ISO 19409:2022 допускает измерение реальной колодки или эквивалентные измерения цифровой 3D-модели. Стандарт также предупреждает, что точки и размеры колодки не обязаны совпадать с анатомическими точками стопы.

### Конструкция

- модель: `derby-low`;
- способ крепления: `cemented`;
- материал верха и подкладки;
- толщина материалов;
- линия шва;
- затяжной припуск;
- количество люверсов;
- эталонная ручная методика.

## Геометрический pipeline

```text
Mesh validation
  -> Last alignment
  -> Landmark verification
  -> Surface curves
  -> Piece segmentation
  -> Triangulated patch extraction
  -> 3D/2D flattening
  -> Distortion map
  -> Seam and allowance generation
  -> Paired-edge validation
  -> Layout and export
```

## Flattening

Криволинейная поверхность колодки не является развёртываемой без искажений. Метод должен:

- сохранять топологию;
- минимизировать локальное растяжение и сжатие;
- учитывать выбранные контрольные длины;
- учитывать направленные свойства материала;
- возвращать карту искажений;
- блокировать результат при превышении согласованного допуска.

Числовой предел допустимого искажения не устанавливается до сравнения с ручной методикой и физическим прототипом.

## Состав деталей первой модели

Предварительный состав требует подтверждения конструктора:

- союзка;
- наружный берец;
- внутренний берец;
- язычок;
- задний ремень/задинка по выбранной конструкции;
- подкладочные детали;
- усилители, если входят в эталонную методику.

## Validation

- mesh имеет миллиметровый масштаб;
- ось и плоскость следа подтверждены;
- линии модели лежат на поверхности;
- поверхности деталей связны;
- 2D-контуры замкнуты;
- карта distortion не содержит недопустимых зон;
- длины сопряжённых швов согласованы;
- отверстия не выходят за безопасные зоны;
- припуски не самопересекаются;
- левая деталь может быть корректно получена зеркалированием только после проверки асимметрии;
- детали помещаются на лист или разбиваются Layout Engine;
- PDF и DXF сохраняют миллиметры.

## Критерий завершения v1

Модуль считается `master-validated`, когда:

1. исходная колодка идентифицирована;
2. эталонная ручная выкройка сохранена;
3. цифровая и ручная выкройки сравнены;
4. пробный верх собран;
5. верх затянут на исходной колодке;
6. отклонения записаны;
7. обувной конструктор подписал протокол;
8. проверенный диапазон параметров зафиксирован.

До выполнения всех пунктов интерфейс не должен предлагать покупку профессиональной обувной выкройки.
